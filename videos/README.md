# Demo video exports

Short **screen recordings** for the course project live here next to this file. The root **[README](../README.md#demo-videos-local-files)** links them by path.

Expected filenames (`.mp4` or change extension in the README if you use something else):

| File | Summary |
|------|---------|
| `act_combined.mp4` | Successful real-robot eval of **ACT**. |
| `diffusion_combined_grasp_portion.mp4` | Successful **diffusion** eval; shows **retry** behavior. |
| `smolvla_eval.mp4` | **SmolVLA** eval: misses grasp, **gripper toggles** repeatedly (notable odd behavior). |
| `simulated_grab.mp4` | **Gazebo** grabs while validating the ML pipeline. |

Videos are usually **too large to commit**; keep them on disk or add selectively. If a link in the main README 404s on GitHub, the file is only on your local tree.
